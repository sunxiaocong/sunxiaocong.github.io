### flume监控
根据你配置的端口地址，flume将会启动一个http服务器。通过http://flume-agent-host:port/metrics访问。  
通过python requests发送post请求
~~~
import requests
url = "http://flume-agent-host:port/metrics"
r = requests.get(url)
info = r.json()
~~~

### 发送钉钉告警
钉钉群是目前工作中一般都会使用的。通过钉钉机器人的接口实现钉钉告警，可以方便实时查看服务状态，避免上线服务异常崩溃。   
我主要监控的channel的状态，主要监控的参数是ChannelSize,ChannelFillPercentage。
~~~
from dingtalkchatbot.chatbot import DingtalkChatbot

# 钉钉群的token
token = "token"
url = "https://oapi.dingtalk.com/robot/send?access_token={token}".format(token=token)
ding = DingtalkChatbot(url)
def send_markdown(size, channel, percentage, host):
    ding.send_markdown(title="flume监控提醒", text="> ### host :{host}\n\n"
                                               "> Channel :{Channel}\n\n"
                                               "> ChannelSize :{ChannelSize}\n\n"
                                               "> ChannelFillPercentage :{ChannelFillPercentage}\n\n"
                       .format(ChannelSize=size, Channel=channel,
                               ChannelFillPercentage=percentage, host=host
                               ))
~~~
至此告警模块就这样了。
